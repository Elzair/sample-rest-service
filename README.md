# sample-rest-service

Service to query a CSV file

## Valid Routes

### /

This route prints out a list of all the items in the CSV file

### /search

This route returns a subset of the items based on certain parameters

#### first_name

the user's first name

#### last_name

the user's last name

#### birth_date_begin

the earliest possible date of birth

#### birth_date_end

the latest possible date of birth

#### favorite_color

the user's favorite color

#### address

the user's address
